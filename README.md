# 🧪 Lab 404 – Plataforma de Cursos Online

**Discord del equipo:** [https://discord.gg/e3d3Udnk](https://discord.gg/e3d3Udnk)

## 🧠 Sobre el proyecto

Este es el proyecto principal de **Lab 404**: una plataforma web para la **venta de cursos online**. Está siendo desarrollado con enfoque profesional y tiene como objetivo construir un producto realista y completo que abarque frontend, backend, base de datos, pagos, roles y gestión educativa.

---

## 🎯 Funcionalidades

- Registro e inicio de sesión para estudiantes y profesores
- Subida y reproducción de videos educativos
- Sistema de compra de cursos con Stripe
- Panel administrativo para gestión de cursos
- Calificación y comentarios
- Roles diferenciados (admin, profesor, estudiante)
- Modo oscuro
- Diseño moderno y responsivo

---

## 🛠️ Stack tecnológico

### Frontend

- React.js (con Vite)
- Tailwind CSS
- React Router
- Axios
- Zustand
- React Hook Form

### Backend

- Node.js + Express
- PostgreSQL
- Prisma (ORM)
- JWT (autenticación)
- Stripe API (pagos)
- Multer (subida de videos)
