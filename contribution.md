# 🧪 Lab 404 – Online Learning Platform

**Discord del equipo:** [https://discord.gg/e3d3Udnk](https://discord.gg/e3d3Udnk)

## 🧠 Sobre el proyecto

Este es el proyecto principal del equipo **Lab 404**. Consiste en una **plataforma web para la venta de cursos online**, desarrollada por un grupo de colaboradores con enfoque profesional. El objetivo es construir un producto realista que incluya backend, frontend, base de datos, pagos y gestión completa, sirviendo como proyecto sólido para nuestros portafolios.

### 🎯 Objetivos principales

- Registro e inicio de sesión para estudiantes y profesores
- Subida y reproducción de videos educativos
- Sistema de compra de cursos con integración de pagos (Stripe)
- Panel administrativo para gestión de cursos
- Calificación y comentarios por parte de usuarios
- Asignación de roles diferenciados (admin, profesor, estudiante)
- Modo oscuro (dark mode)
- Diseño moderno y responsivo

---

## 🛠️ Stack tecnológico

### Frontend

- React.js (con Vite)
- Tailwind CSS
- React Router
- Axios
- Zustand
- React Hook Form

### Backend

- Node.js + Express
- PostgreSQL
- Prisma (ORM)
- JWT (autenticación)
- Stripe API (pagos)
- Multer (subida de videos)

## 📂 Guía de ramas

Todos los commits deben seguir esta convención de ramas para mantener el orden en el proyecto:

- 🚧 Para nuevas funcionalidades: `features/nombre-de-la-funcionalidad`
- 🛠️ Para correcciones rápidas: `hotfix/nombre-del-fix`

> Una vez completados los cambios, se debe crear un **pull request** hacia la rama principal (`main`).
